Math Combat Game

The Math Combat Game is a simple yet engaging console-based game made in C language. In this game, two players are allowed to fight against each other by answering math questions. Every time an answer is correct, the player gains points which increases their scores, and every wrong answer reduces the player's health. The game has different levels of difficulties for players to encounter more complex questions. This game has login, registration and colored outputs which make this game different from other console-based games and looks fun to play
